Github
======

This repository is implemented the following repository with Elixir.
https://github.com/KazuCocoa/sampleGitHubWebhooks


## How to use

```
$ git clone git@github.com:KazuCocoa/elixir-github-api.git
$ cd elixir-github-api
$ mix deps.get
$ iex -S mix
```
